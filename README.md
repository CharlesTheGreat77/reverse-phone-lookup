Reverse Phone Lookup OSINT with python3 and playwright

# Install
```
pip3 install -r requirements.txt
playwright install
```

# Optional (venv)
```
python3 -m venv <venv_name>
source <venv_name>/bin/activate
pip3 install -r requirements.txt
playwright install
```

# Usage
```
usage: python3 lookup.py 9998883333
```
